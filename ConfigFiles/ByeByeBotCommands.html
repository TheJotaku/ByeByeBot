ByeByeBot

The main purpose of this bot was to send a message in a text channel tagging a user whenever they were to leave a voice channel. This messsage is simply asking if the user has said "bye" to everyone before disconnecting.

Upon developing this bot other features have been added through commands. The commands can be found below.


All Commands Must Have A Prefix of !

Commands Description
TellByeBye Bot will display a document with all commands.
Hello Bot responds with "You mean goodbye, right?"
FuckYou Bot will disconnect the user from the current voice channel and then send a message in a text channel tagging the user.
TheByeByeBot Bot responds with "Don't Think It! Don't Say It!"
ByeByeBye Bot responds with "You may hate me but it ain't no lie Baby bye bye bye"

This is just a quick little project nothing fancy. Hope you have fun with it! 😄
